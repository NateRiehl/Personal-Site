Festival By the Sea

I completed this project while an intern at NPND, Inc. I developed a website that uses an animated map to dynamically provide information about a summer festival called Festival by the Sea Check Out the Source Code

I implemented the Leaflet.js open source library and then used asynchronous Javascript and GeoJSON to access, filter, and enhance geographical data about the event. In the next stage, we used the PhoneGap framework to convert the website to Android/IOS applications. We also added push notifications to supply users with information about ongoing music and deals and implemented features like live “crowd viewing” and online vendor signup to the mobile and web applications.

Finally, I presented our product by myself at a town merchant meeting where the applications received widespread approval. The website and mobile applications went on to garner substantial use by thousands of vendors and attendees!

  • Client: Non-Profit
  • Date: May - August 2015
  • Service: Web Development
